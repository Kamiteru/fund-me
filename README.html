Quickstart

Testing

forge test

Deployment

Set environment variables (e.g. in .env):

  • SEPOLIA_RPC_URL
  • PRIVATE_KEY
  • Optional: ETHERSCAN_API_KEY

Deploy:

forge script script/DeployFundMe.s.sol \
  --rpc-url $SEPOLIA_RPC_URL \
  --private-key $PRIVATE_KEY \
  --broadcast \
  --verify \
  --etherscan-api-key $ETHERSCAN_API_KEY

Interaction

cast send <FUNDME_CONTRACT_ADDRESS> "fund()" --value 0.1ether --private-key $PRIVATE_KEY

forge script script/Interaction.s.sol:FundFundMe --rpc-url $SEPOLIA_RPC_URL --private-key $PRIVATE_KEY --broadcast
forge script script/Interaction.s.sol:WithdrawFundMe --rpc-url $SEPOLIA_RPC_URL --private-key $PRIVATE_KEY --broadcast
